**`src/client/components/AppButton.vue`:**

Props:
- `variant`: `'primary' | 'secondary' | 'ghost'` (default: `'primary'`)
- `size`: `'sm' | 'md' | 'lg'` (default: `'md'`)
- `disabled`: Boolean
- `loading`: Boolean

Uses a native `<button>` element. Styling rules:
- Primary variant: `background: var(--color-primary)`, `color: var(--color-navy)` (dark text on green)
- Secondary variant: `background: var(--color-bg-secondary)`, `border: 1px solid var(--color-border)`, `color: var(--color-body-text)`
- Ghost variant: `background: transparent`, `color: var(--color-link)`
- Disabled: `opacity: 0.5; cursor: not-allowed; pointer-events: none`
- Loading: Show a small CSS spinner (rotating border trick) inline before the slot text. Disable pointer events while loading.
- Sizes: sm = `padding: var(--spacing-xs) var(--spacing-sm); font-size: var(--font-size-sm)`, md = `padding: var(--spacing-sm) var(--spacing-md); font-size: var(--font-size-base)`, lg = `padding: var(--spacing-sm) var(--spacing-lg); font-size: var(--font-size-lg)`
- All variants: `border-radius: var(--radius-md)`, `font-family: var(--font-family)`, `transition: background var(--transition-fast)`, `cursor: pointer`
- The `disabled` and `loading` props both set `aria-disabled="true"` on the button element

**`src/client/components/AppBadge.vue`:**

Props:
- `variant`: `'default' | 'success' | 'danger' | 'muted'` (default: `'default'`)

Renders an inline `<span>` element. Styling:
- Default: `background: color-mix(in srgb, var(--color-primary) 20%, transparent)`, `color: var(--color-navy)`
- Success: `background: color-mix(in srgb, var(--color-success) 15%, transparent)`, `color: var(--color-success)`
- Danger: `background: color-mix(in srgb, var(--color-danger) 15%, transparent)`, `color: var(--color-danger)`
- Muted: `background: var(--color-bg-secondary)`, `color: var(--color-muted)`
- All: `padding: 2px var(--spacing-sm)`, `border-radius: var(--radius-lg)`, `font-size: var(--font-size-xs)`, `font-weight: 500`, `display: inline-flex`, `align-items: center`

**`src/client/components/AppCheckbox.vue`:**

Props:
- `modelValue`: Boolean
- `label`: String
- `disabled`: Boolean

Emits: `update:modelValue`

Uses Reka UI: `import { CheckboxRoot, CheckboxIndicator } from 'reka-ui'`

Structure from research (Pattern 5): `<label>` wrapping `<CheckboxRoot>` with `<CheckboxIndicator>` containing an SVG checkmark. Style using `[data-state="checked"]` and `[data-disabled]` CSS attribute selectors.

Checkbox box: 18px square, `border: 2px solid var(--color-border)`, `border-radius: var(--radius-sm)`, `background: var(--color-bg)`.
Checked state: `background: var(--color-primary)`, `border-color: var(--color-primary)`.
Disabled: `opacity: 0.4`, `cursor: not-allowed`.
Label: `margin-left: var(--spacing-sm)`, `color: var(--color-body-text)`, `font-size: var(--font-size-base)`.

**`src/client/components/AppTooltip.vue`:**

Props:
- `content`: String
- `side`: `'top' | 'bottom' | 'left' | 'right'` (default: `'top'`)

Uses Reka UI: `TooltipProvider, TooltipRoot, TooltipTrigger, TooltipPortal, TooltipContent, TooltipArrow`

Structure from research (Pattern 6): Provider wraps Root which contains Trigger (with `as-child` and a default `<slot>`) and Portal containing Content + Arrow.

Tooltip content styling: `background: var(--color-navy)`, `color: #fff`, `padding: var(--spacing-xs) var(--spacing-sm)`, `border-radius: var(--radius-sm)`, `font-size: var(--font-size-sm)`, `z-index: 1000`, `max-width: 250px`.
Arrow: `fill: var(--color-navy)`.
Provider delay: 300ms.

Add Reka UI's built-in animation attributes: `data-state="delayed-open"` for enter, `data-state="closed"` for exit. Add CSS transitions for opacity.

**`src/client/components/AppProgressBar.vue`:**

Props:
- `value`: Number (default: 0)
- `max`: Number (default: 100)
- `indeterminate`: Boolean (default: false)

Uses Reka UI: `import { ProgressRoot, ProgressIndicator } from 'reka-ui'`

Structure from research: `<ProgressRoot>` wrapping `<ProgressIndicator>`.

Root styling: `height: 8px`, `background: var(--color-bg-secondary)`, `border-radius: var(--radius-lg)`, `overflow: hidden`, `width: 100%`.
Indicator: `height: 100%`, `background: var(--color-primary)`, `border-radius: var(--radius-lg)`, `transition: transform var(--transition-normal)`.
Transform: `translateX(-${100 - (value / max) * 100}%)` (when not indeterminate).
Indeterminate: Use a CSS keyframe animation (`@keyframes indeterminate`) that slides the indicator back and forth. Set indicator width to 40% and animate `translateX` from -100% to 200%.
