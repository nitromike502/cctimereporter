<task type="auto">
  <name>Task 1: Switch triggerImport to EventSource with progress tracking</name>
  <files>src/client/pages/TimelinePage.vue</files>
  <action>
1. Add a new ref: `const importProgress = ref({ processed: 0, total: 0 })`
2. Add a ref to track the EventSource for cleanup: `const importEventSource = ref(null)`

3. Rewrite `triggerImport()`:
   - Guard: if `importRunning.value` return.
   - Set `importRunning.value = true` and reset `importProgress.value = { processed: 0, total: 0 }`.
   - Create `const source = new EventSource('/api/import/progress')`.
   - Store in `importEventSource.value = source`.
   - Add `source.addEventListener('progress', (e) => { importProgress.value = JSON.parse(e.data) })`.
   - Add `source.addEventListener('complete', (e) => { source.close(); importEventSource.value = null; importRunning.value = false; fetchTimeline(); })`.
   - Add `source.addEventListener('error', () => { source.close(); importEventSource.value = null; importRunning.value = false; error.value = 'Import failed'; })`.
   - Note: Do NOT await anything — EventSource is event-driven, the function returns immediately.

4. Add cleanup on unmount: `import { onUnmounted } from 'vue'` (add to existing import). Add `onUnmounted(() => { importEventSource.value?.close() })`.

5. Pass `importProgress` to TimelineToolbar as a new prop:
   `:import-progress="importProgress"`
  </action>
  <verify>Run `npm run build` to confirm no compilation errors.</verify>
  <done>triggerImport uses EventSource to GET /api/import/progress, updates importProgress ref in real time, cleans up on unmount</done>
</task>

2. Replace the progress-container div (currently `<div v-if="importRunning" class="progress-container">`) with:
   ```html
   <div v-if="importRunning" class="progress-container">
     <AppProgressBar
       :value="props.importProgress.processed"
       :max="props.importProgress.total || 1"
       :indeterminate="props.importProgress.total === 0"
     />
     <span v-if="props.importProgress.total > 0" class="progress-text">
       {{ props.importProgress.processed }} / {{ props.importProgress.total }}
     </span>
   </div>
   ```
   This shows indeterminate mode until the first progress event arrives (total=0 initially), then switches to determinate with a text count.
