    3. Compute the cutoff date string (ISO8601, lexicographic comparison):
       ```js

         ? new Date(Date.now() - maxAgeDays * 24 * 60 * 60 * 1000).toISOString()

       ```

    4. Replace `const importedSizes = getImportedFileSizes(db)` with:

    5. Replace the existing 3-line skip loop (lines ~297-303) with the multi-tier filter chain. The loop iterates `files` and classifies each into `toImport` or skip:

       ```js
       for (const file of files) {
         const cached = importedFileInfo.get(file.path);

         // Skip 1: size unchanged (existing behavior, fast path)
         if (!force && cached?.fileSize === file.size) {
           toSkip.push(file);
           continue;
         }

         // Skip 2: rolling window — cached lastMessageAt is before cutoff
         if (!force && cutoffDate && cached?.lastMessageAt && cached.lastMessageAt < cutoffDate) {
           toSkip.push(file);
           continue;
         }

         // Skip 3: new file (no cache) — peek first timestamp
         if (!force && cutoffDate && !cached) {
           const firstTs = peekFirstTimestamp(file.path);
           if (firstTs && firstTs < cutoffDate) {
             // Record as skipped_old so subsequent imports don't re-peek (IMP-02)
             updateImportLog(db, file.path, file.sessionId, file.size, 'skipped_old', null, firstTs, firstTs);
             filesSkipped++;
             continue;
           }
         }

         toImport.push(file);
       }
       ```

    6. Update the agent file skip check (around line ~333) similarly: replace `importedSizes.get(agentFile.path) === agentFile.size` with `importedFileInfo.get(agentFile.path)?.fileSize === agentFile.size`.

    7. In the `importFile()` success path (line ~251), update the `updateImportLog` call to pass timestamps:
       ```js
       updateImportLog(db, file.path, file.sessionId, file.size, 'ok', null, firstMessageAt, lastMessageAt);
       ```
       The `firstMessageAt` and `lastMessageAt` variables are already computed at lines ~192-193 inside `importFile()`.

         if (importRunning) {

           return { error: 'Import already in progress' };
         }

         try {

         } finally {
           importRunning = false;

       });
