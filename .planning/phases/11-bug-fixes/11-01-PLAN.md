In `src/importer/parser.js`, add a regex constant near the top of the file (after the imports, before `parseTranscript`):

```js
// Patterns for synthetic/system-generated user messages that should not be captured as firstPrompt.
// These are user-type messages with isMeta=undefined that contain XML command structures.
const SYNTHETIC_MSG_RE = /^\s*(<command-name>|<teammate-message|<local-command)/;
```

Then modify the firstPrompt capture block (lines 92-98) to check the regex before storing:

Change from:
```js
if (!firstPrompt && msg.type === 'user' && !msg.isMeta) {
  const text = extractContentText(msg);
  const trimmed = text?.trim();
  if (trimmed) {
    firstPrompt = trimmed.slice(0, 200);
  }
}
```

To:
```js
if (!firstPrompt && msg.type === 'user' && !msg.isMeta) {
  const text = extractContentText(msg);
  const trimmed = text?.trim();
  if (trimmed && !SYNTHETIC_MSG_RE.test(trimmed)) {
    firstPrompt = trimmed.slice(0, 200);
  }
}
```

The only change is adding `&& !SYNTHETIC_MSG_RE.test(trimmed)` to the inner if condition.

  <action>
In `src/importer/index.js`, add a regex constant near the top of the file (after imports, near other constants):

```js
// Worktree-based subagent project path patterns.
// Claude Code's EnterWorktree creates projects at paths like:
//   /home/user/project/.claude/worktrees/tmp-pr-review-abc123/
// These are registered in ~/.claude.json as separate projects with -tmp- in the path.
const WORKTREE_PROJECT_RE = /\/-tmp-|\/\.claude\/worktrees\//;
```

**Important:** `importFile()` does NOT have access to the `project` object — its signature is `importFile(db, file, projectId, options, sessionIndex)`. The `project` object is only available in `importAll()`. So the worktree check must be computed in `importAll()` and passed through `options`.

**Step A — In `importAll()`, compute `isWorktreeProject` before calling `importFile()`.**

Find the call site at line 354 inside the `for (const project of projects)` loop:

```js
const result = await importFile(db, file, projectId, { verbose }, sessionIndex);
```

Replace with:

```js
const isWorktreeProject = WORKTREE_PROJECT_RE.test(project.projectPath);
const result = await importFile(db, file, projectId, { verbose, isWorktreeProject }, sessionIndex);
```

**Step B — In `importFile()`, extract `isWorktreeProject` from options and combine with team-based detection.**

Find the destructuring at line 170:

```js
const { verbose } = options;
```

Replace with:

```js
const { verbose, isWorktreeProject = false } = options;
```

Then find the existing subagent detection at line 206:

```js
const isSubagent = data.userType === 'external' && data.agentName != null;
```

Replace with:

```js
// Pattern B: team-based subagents (userType=external + agentName set)
// Pattern C: worktree-based subagent projects (-tmp- or .claude/worktrees/ in path)
const isTeamSubagent = data.userType === 'external' && data.agentName != null;
const isSubagent = isTeamSubagent || isWorktreeProject;
```
