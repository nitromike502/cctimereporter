In `src/importer/ticket-scorer.js`, make the following changes:

**1. Update TICKET_PATTERN regex.**

Change from:
```js
export const TICKET_PATTERN = /[a-zA-Z]{2,8}-\d+/gi;
```

To:
```js
export const TICKET_PATTERN = /\b[A-Z]{2,8}-\d{1,6}\b/gi;
```

Changes:
- Added `\b` word boundary assertions to prevent mid-word matches (e.g. ICATIONS-671)
- Changed `\d+` to `\d{1,6}` to reject timestamp-suffixed IDs (e.g. APPROVAL-1772078192534)
- Changed `[a-zA-Z]` to `[A-Z]` since the `i` flag already makes it case-insensitive (cleaner)

**2. Expand TICKET_PREFIX_DENYLIST.**

Change from:
```js
export const TICKET_PREFIX_DENYLIST = new Set(['SHUTDOWN']);
```

To:
```js
export const TICKET_PREFIX_DENYLIST = new Set([
  // Auto-generated IDs (timestamp or sequential suffixes)
  'SHUTDOWN', 'APPROVAL', 'BASH',
  // AI model names
  'CLAUDE', 'OPUS', 'GEMINI', 'GPT', 'SONNET', 'HAIKU',
  // CSS / design tokens
  'GRAY', 'GREY', 'RED', 'GREEN', 'BLUE', 'ORANGE', 'YELLOW', 'PURPLE', 'PINK', 'WHITE', 'BLACK',
  'PRIMARY', 'SECONDARY', 'SURFACE', 'ACCENT', 'NEUTRAL',
  // Standards, encodings, tooling
  'UTF', 'ISO', 'PSR', 'WSL', 'RFC', 'HTTP', 'SHA', 'MD',
  // Framework / language names
  'VUE', 'REACT', 'NODE', 'LARAVEL', 'CPYTHON', 'PYTHON', 'RUBY',
  // Version identifiers
  'VERSION', 'RELEASE',
  // Placeholder / example ticket prefixes (documentation artifacts)
  'TICKET', 'ABC', 'DEF', 'USER', 'PLAYER', 'TEAM', 'TEST', 'EXAMPLE', 'SAMPLE', 'DEMO',
]);
```

IMPORTANT: Do NOT add STORY, BUG, TASK, EPIC, FEATURE, or ISSUE to the denylist. These are legitimate ticket prefixes used by real project management systems.

In `src/importer/ticket-scorer.js`, add a minimum score constant after the existing constants (near `TICKET_PREFIX_DENYLIST`):

```js
// Minimum score a ticket must achieve to be considered the primary ticket.
// Filters single-mention noise: a ticket mentioned once in content scores 10 pts (below threshold).
// A ticket mentioned twice (20 pts) or once in content + once in branch (110+ pts) passes.
export const MIN_TICKET_SCORE = 15;
```

Then modify the return logic at the end of `scoreTickets()`. Change from:

```js
if (ticketScores.size === 0) return null;

// Return highest-scoring ticket
let best = null;
let bestScore = -Infinity;
for (const [ticket, score] of ticketScores) {
  if (score > bestScore) {
    bestScore = score;
    best = ticket;
  }
}

return best;
```

To:

```js
if (ticketScores.size === 0) return null;

// Return highest-scoring ticket, if it meets the minimum threshold
let best = null;
let bestScore = -Infinity;
for (const [ticket, score] of ticketScores) {
  if (score > bestScore) {
    bestScore = score;
    best = ticket;
  }
}

return bestScore >= MIN_TICKET_SCORE ? best : null;
```

The only change is the final return line: `return bestScore >= MIN_TICKET_SCORE ? best : null;`
