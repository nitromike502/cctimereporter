## Milestone Summary

**Key Decisions:**
- Rolling 30-day import window with peek-and-skip for instant re-skip
- sessions-index.json is INSIDE transcriptDir (not parent dir)
- firstPrompt truncated to 200 chars in JSONL extraction
- DaySummary receives timelineData.projects (unfiltered) for accurate totals
- Dark mode uses [data-theme='dark'] on documentElement with user toggle
- Tour fires only when data.projects.length > 0
- SYNTHETIC_MSG_RE filters slash command XML from firstPrompt capture
- Worktree project detection via path pattern /-tmp-/ or /.claude/worktrees/
- Ticket denylist expanded to 35+ prefixes, MIN_TICKET_SCORE = 15

**Issues Resolved:**
- Session summaries showing slash command XML instead of real user text
- Worktree-based subagent sessions polluting timeline
- DaySummary table column misalignment
- Ticket/Branch tabs missing project context
- Hundreds of false-positive ticket detections (OPUS-4, GRAY-100, UTF-8, etc.)

**Issues Deferred:**
- Subagent working time attribution to parent session (tracked, not implemented)
- UI for reviewing/correcting ticket assignments via bulk DB updates

**Technical Debt Incurred:**
- None significant
